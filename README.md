# Initial Setup

This is an app created with react that connects to the [Chuck Norris Jokes API](https://api.chucknorris.io/).

## Installation

Use the package manager [npm](https://www.npmjs.com/) for install the dependencies.

```
npm install
```

## Usage

```
npm start
```
